# Privacy Policy – PDI Trips Exporter

**Last updated:** 20 January 2026

## Overview

PDI Trips Exporter is a Chrome extension that allows authorised users to export trip scheduling data from the PDI PLC web application into an Excel file. The extension operates only on supported PDI PLC domains and runs entirely within the user’s browser.

This privacy policy explains what data the extension accesses, how that data is used, and how it is handled.

## Data Accessed

The extension accesses the following types of data:

- **Website content** displayed within the PDI PLC web application, including trip, delivery, and scheduling information selected by the user.
- **Session authentication information** (such as an authorisation token and tenant identifier) that is required to authenticate requests to PDI scheduling APIs during a user-initiated export.

The extension does **not** access or collect:
- Usernames or passwords
- Personally identifiable information unrelated to trip data
- Financial or payment information
- Health information
- Personal communications
- Browsing history outside supported PDI PLC domains

## How Data Is Used

Data accessed by the extension is used solely to:

- Retrieve trip and delivery data corresponding to the date range currently selected in the PDI PLC interface.
- Generate an Excel file containing that data.
- Download the generated file to the user’s computer when the user explicitly clicks the Export button.

All processing occurs locally within the user’s browser.

## Data Storage and Retention

- Session authentication information is stored **locally** in the browser only when required for a user-initiated export.
- This information is temporary, expires automatically after a short period, and is cleared after the export process completes.
- The extension does not permanently store trip data, exported files, or authentication information.

## Data Sharing

- The extension does **not** sell, share, or transfer user data to third parties.
- Data is transmitted only to PDI scheduling API endpoints required to retrieve trip information for export.
- No analytics, tracking, or advertising services are used.

## User Control

- The extension performs no background data collection.
- All data access and exports are initiated explicitly by the user.
- The extension operates only on supported PDI PLC domains and does not run on other websites.

## Security

The extension follows the principle of least privilege:
- It requests only the permissions required to perform its single purpose.
- It limits its operation to specific PDI PLC domains.
- It does not execute remote code or dynamically load scripts from external sources.

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.
